## 📌 프로젝트 개요
- Ghostouch는 스마트폰에서 손 제스처만으로 기기 제어 및 앱 기능을 간편하게 제어할 수 있는 모바일 어플리케이션입니다. 스마트폰 전면 카메라와 Mediapipe Hand Detection 모델을 이용해 손 모양을 빠르고 정확하게 인식하고 네트워크 없이 실행이 가능하며 사용자가 원하는 제스처를 제어 기능에 연결할 수 있어 누구나 쉽게 사용할 수 있습니다.
<details>
  <summary><b>더보기</b></summary>
  
  <a id="핵심-기능"></a>
  ### ⚡ 핵심 기능
  
  1. **실시간 제스처 인식**  
      - 전면 카메라와 Mediapipe Hand Detection을 활용하여 손 모양과 움직임을 빠르게 인식
  
  2. **사용자 지정 제스처 매핑**  
      - 원하는 제스처(엄지척)를 특정 기능(앱 실행)에 직접 연결 가능
  
  3. **백그라운드 실행**  
      - 앱을 최소화해도 제스처 인식이 계속 동작하며, 실행 시간을 원하는 대로 설정
  
  4. **배터리 절약**  
      - 필요할 때만 동작하도록 설정해 불필요한 자원 사용을 줄이고 배터리를 효율적으로 관리
  
  5. **온디바이스 처리**  
      - 모든 연산을 스마트폰 내에서 처리해 인터넷 연결이 필요 없으며, 개인정보도 안전하게 보호
  
  6. **증분 학습 지원**  
      - 기존 제스처와 새 제스처를 함께 학습할 수 있고, 새로운 사용자 정의 제스처를 자유롭게 추가 가능
