# Ghostouch Flutter 프로젝트

## 시작하기

아래 순서대로 명령어를 실행하세요.

1. 프로젝트 정리

   ```bash
   flutter clean
   ```

2. 패키지 설치

   ```bash
   flutter pub get
   ```

3. 앱 실행
   ```bash
   flutter run
   ```
